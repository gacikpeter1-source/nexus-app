## âœ¨ Animations & Effects

### 1. Background Animation
```
Slow-moving gradient orbs
Pulsing opacity (8s cycle)
Creates depth and energy
```

### 2. Card Hover Effects
```
Before hover: flat, slight transparency
On hover:     lifts up 4px
              shadow intensifies
              border glows
              shine effect sweeps across
```

### 3. Navigation Underlines
```
Default:   no underline
Hover:     line grows from left to right
Duration:  300ms smooth
```

### 4. Button Animations
```
Click:     slight scale down
Release:   bounces back up
Hover:     lifts up, shadow grows
          background intensifies
```

### 5. Page Transitions
```
Enter:     fade in + slide up
Duration:  800ms
Stagger:   elements appear sequentially
```
